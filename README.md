# Sales Performance Dashboard — Excel

## Overview

Analyzed 1,000+ real-world gift delivery orders across 

Indian cities using Microsoft Excel.

## Tools Used

Microsoft Excel — Pivot Tables, Slicers, VLOOKUP, Charts

## Key Insights

- Revenue grew from ₹28,983 (April) to ₹62,662 (June)

- Soft Toys was the top performing category at ₹66,581

- Peak order hours and high-revenue cities identified

- Built interactive dashboard with KPI summaries
